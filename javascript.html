JavaScript

JavaScript je programovací jazyk, ktorý sa podobne ako kaskádové štýly vkladajú do HTML kódu. Pochádza z dolnej spoločnosti Nescape. Patrí do rodiny jazykov C, C++ a Java. Je to klientovský skript, to znamená, že sa program odosiela zo serveru do počítača a potom tam je vykonaný, narozdiel od serverových skriptov, ako PHP, ktoré sa prevedú a užívatelovi zobrazí iba výsledok.

Premenné

Premenna je v programovaní označenie pre identifikátor, ktorý uschováva určitú informaciu počas behu programu. Je to reprezentacia miesta v pamäti.

Deklarácia premennej.

var identifikator;
var cislo = 5;
var veta = "Toto je veta.";
var pi = 3.14;
