Textový editor je softvér určený na vytváranie a úpravu textových dokumentov. Na rozdiel od textových procesorov, s ktorými sa často mylne zamieňajú, im chýba schopnosť meniť vzhľad dokumentu. Nie sú totiž orientované na DTP, ani ako WYSIWYG editory pre WWW, používajú sa ale napríklad ako súčasti vývojových prostredí programátorov.

Úvod

Zdrojový kód webových stránok je možné písať v jednoduchých textových editoroch. Existujú ale mnohé programy ktoré veľmi uľahčujú prácu. Obsahujú funkcie priamo vytvoréné pre vývojárov. Stránka StackOverflow vydala prieskum v skorom zoradila editory používané pri vývoji webu na základe popularity. Z týchto výsledkov vyberieme tie najzaujímavejšie.

Notepad++

Notepad++ je freewareový textový editor a editor zdrojových kódov pre platformu Microsoft Windows. Projekt sa nachádza na SourceForge.net odkiaľ už bol stiahnutý viac ako sedemnásť miliónovkrát. Tento projekt, založený na scintilla editor, je napísaný v jazyku C++. Cieľom programu Notepad++ je ponúknuť jednoduchý a efektívny spôsob úpravy kódov s úplne upraviteľnými GUI.

Sublime Text

Sublime Text je multiplatformný editor textových súborov a zdrojového kódu písaný v C++ s programovým API v Pythone. Jeho funkcionalita je rozšíriteľná pluginmi vytváranými komunitou okolo Sublime Textu. Samotný editor nie je open source ani zadarmo, väčšina pluginov však áno.

Vim

Vim je open source textový editor, ktorý je možné spustiť v prostredí väčšiny operačných systémov. Je obľúbený hlavne medzi skúsenými užívateľmi operačných systémov unixového typu. Okrem klasického Vimu existuje celá rada editorov, ktoré sú založené na princípe Vimu, ale majú nejaké špecifické vlastnosti napr. Yzis pre prostredie KDE.

Emacs

GNU Emacs je textový editor, ktorého počiatky siahajú do 70. rokov 20. storočia. Do dnešného dňa se stále vyvíja a vydávajú se nové verzie. Patrí mezi jeden z nejobľúbenejších textových editorov platformy GNU/Linux. Aktuálna verzia obsahuje i grafické prostredie, čo uľahčuje prácu s editorom hlavne jeho začínajúcím používateľom. Emacs je populárny pre svoju prispôsobiteľnosť a rozšíriteľnosť; obsahuje interpreter programovacieho jazyka LISP, pomocou kterého je možné vytvárať moduly s novými funkciami. Existují rozšírenia, které umožňujú používať Emacs ako webový prehliadač, e-mailový klient a pod.
